Meu primeiro contato com o Git.
Comandos do Git

Status Geral
git status

Trackear
git add -A

Comitar
git commit -m "mensagem"

Ver situação do Repositório
git log

Voltar para etapa desejada do desenvolvimento
git reset --hard hash
git reset --soft hash

Movimentar entre Branch
git branch <nome branch>

Mudar de Branch
git checkout <nome branch>
